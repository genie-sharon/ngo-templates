# Kindonar Product Roadmap

## Current Release: v1.3 — AI Experience Layer (Q2 2026)

The AI Experience Layer delivers 14 AI service modules across content generation, optimization, and assessment. This release establishes Kindonar as an intelligent NGO website builder that actively assists users in creating compelling, accessible, and effective websites.

### AI Service Modules
- Content generation tuned for non-profit messaging
- Smart layout and theme recommendations
- SEO optimization and accessibility auditing
- Donation psychology and storytelling enhancement

## Next Up: v1.4 — Multi-Language & Donations (Q3 2026)

### Multi-Language Support
- i18n framework with next-intl
- 10+ language translations (UN languages + regional)
- Automated translation workflow via AI
- RTL layout support for Arabic and Hebrew
- Language switcher component
- SEO hreflang tag generation
- Per-language content management in builder

### Donation Gateway Integration
- Stripe, PayPal, and GoFundMe integration
- Embedded donation forms with builder customization
- Recurring donation support
- Donation progress bars and goal tracking
- Tax receipt generation
- Donor management dashboard
- Payment processing analytics

### Builder Enhancements
- Undo/redo with slider
- Global section styles
- Custom CSS injection per page
- Section copy/paste between pages

## Future: v2.0 — Collaboration & Marketplace (Q1 2027)

### Collaborative Editing
- Multi-user real-time editing with WebSockets
- Role-based access (Admin, Editor, Reviewer)
- Change tracking and approval workflows
- Comment and annotation system per section
- Version history with diff comparison
- Audit logging for compliance

### CMS Integration
- Headless CMS connectors (Strapi, Sanity, Contentful)
- Dynamic content binding from external APIs
- Automated content syncing
- Media library with DAM integration
- Blog/news engine with RSS
- Form submissions storage

### Marketplace
- Community template marketplace
- Custom theme publishing
- Third-party section plugins
- AI prompt packs (language-specific, cause-specific)
- Asset marketplace (icons, illustrations, stock photos)
- Revenue sharing for creators

### Platform Expansion
- White-label solution for NGOs
- Self-hosted deployment option
- Mobile app for on-the-go editing
- API for programmatic site management
- Webhooks and automation triggers
- Advanced analytics dashboard

## Vision: v3.0 — Intelligent Impact Platform (H2 2027)

- **AI Site Generator**: Build a complete site from a single mission statement
- **Impact Dashboard**: Real-time donor/volunteer analytics integrated into website
- **Smart Personalization**: Dynamic content based on visitor segment
- **Automated Campaigns**: AI-driven fundraising campaign creation
- **Community Hub**: Member portal, forums, and event management
- **Global Scale**: 50+ languages, multi-currency, regional compliance
